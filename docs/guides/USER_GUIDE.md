# AppForge User Guide

Welcome to AppForge! This guide explains how to use the applications generated by your administrators.

## Dashboard Navigation

- **Command Palette**: Press `Cmd+K` (Mac) or `Ctrl+K` (Windows) from anywhere to instantly search for pages, models, or data.
- **Sidebar**: Use the left sidebar to navigate between your different modules.
- **Language Switcher**: Click the flag icon in the top right header to switch between English, Hindi, and Kannada.

## Working Offline (PWA)

AppForge is a Progressive Web App (PWA). You can install it to your desktop or home screen!
1. **No Internet? No Problem**: If you lose connection on a flight or subway, the app will continue to work.
2. **Offline Saves**: You can continue to fill out forms and click "Save". A yellow banner will appear letting you know your data is queued locally.
3. **Automatic Sync**: The moment your internet connection is restored, the banner will spin green and automatically sync your queued data to the cloud. You don't have to do anything!

## Importing Data via CSV

1. Navigate to the module you want to import data into.
2. Click **Import CSV**.
3. **Step 1**: Drag and drop your `.csv` file.
4. **Step 2**: The system will attempt to auto-map your CSV columns (e.g., "Full Name") to the system fields (e.g., "firstName"). Adjust these if they are incorrect.
5. **Step 3**: Review the preview grid. The system will warn you if you have text in a number column *before* you import.
6. **Step 4**: Click Import. A progress bar will appear. If a few rows fail (e.g., bad email format), the system will successfully import the rest and provide you with an Error Log showing exactly which rows failed so you can fix them.
